National unit GLBs (optional CC0 catalog)
==========================================

Default armies use YOUR Hunyuan meshes:

  public/models/rigged/{region}_{team}.glb
  public/models/rigged/{region}_{team}_{piece}.glb   (per-piece bake, preferred)

Optional team-agnostic silhouettes from GitHub CC0:

  public/models/units/{region}/{piece}.glb

  npm run free-units:national

Honest limit: 72 unique rigged, nation-themed humanoids per chess role are not
available free on GitHub. The national download script uses curated DISTINCT
props/deities/creatures from ToxSam/cc0-models-Polygonal-Mind (CC0) as placeholders
until you regenerate per-piece meshes from PNG crops (Hunyuan / Tripo).

KayKit Adventurers are NOT used as nation stand-ins.
